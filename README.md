# Boikhutso Mashigo - Professional Portfolio
 
This is the official portfolio website for Boikhutso Mashigo, a Software Engineer specializing in Data Engineering. The site serves as a central hub to showcase professional projects, technical skills, and certifications.

 ## Key Features

    Modern UI/UX: Clean, professional design using Glassmorphism effects and a monochrome color palette.

    Responsive Design: Fully optimized for desktops, tablets, and mobile devices using CSS Grid and Flexbox.

    Dynamic Navigation: Smooth scrolling and a mobile-friendly hamburger menu.

    Project Showcase: Dedicated sections for key projects like the Food Delivery Management System and Brownfields Legacy Optimization.

## Tech Stack

    Frontend: HTML5, CSS3 (Custom Variables, Flexbox/Grid), JavaScript (ES6+)

    Hosting: Netlify (Continuous Deployment via GitHub)

    Version Control: Git & GitHub
